# Just Do It – To-Do List App

A clean and interactive To-Do List web app built with HTML, CSS, and JavaScript.
Add, delete, and manage your daily tasks with smooth fall animations and localStorage to keep your data even after refreshing the page.

# Features

 Add and remove tasks dynamically

 Fall animation when deleting a task

 Saves tasks automatically using localStorage

 Simple, responsive, and modern design

 Built with pure JavaScript (no frameworks)

# How It Works 

Type your task in the input field and hit Add.

Each task appears in the list instantly.

Click the Delete button to trigger a fall animation and remove the task.

All tasks are stored in your browser, so they stay even after you reload the page!

# Tech Stack

HTML5 – Structure

CSS3 – Styling & animations

JavaScript (ES6) – Functionality & logic

LocalStorage – Save user data persistently
